Adobe XD

Saya memiliki pengalaman dalam merancang antarmuka pengguna (UI) yang intuitif dan pengalaman pengguna (UX) yang menarik menggunakan Adobe XD. Proses desain ini membantu saya mengembangkan prototipe interaktif yang memungkinkan klien dan tim untuk memahami dan menguji alur kerja aplikasi sebelum pengembangan dimulai.

Tailwind css

Saya sangat menguasai Tailwind CSS, sebuah framework CSS utility-first yang memungkinkan saya untuk dengan cepat merancang dan mengimplementasikan desain yang konsisten dan efisien.
